# Show Me The Fruit App

Compatible with iOS version 13.0 and above. 

To refresh data pull down the fruit list screen.

Design pattern: MVP

IDE: Xcode 11.2

Language: Swift 5

Development Techniques:

- TDD
- Unit testing 
- UI testing
- Object mocking
- Protocols to reduce direct dependancy
